Neural Network Components — Definitions & Roles

COMPONENT 01

Layers

Layers are groups of neurons arranged in stages within a neural network. They help process information step-by-step as data moves through the model.

Types of Layers:

  • Input Layer: Receives the raw data, such as images or numerical values.
  • Hidden Layers: Perform calculations and detect patterns in the data.
  • Output Layer: Produces the final prediction or classification result.
Role: Layers allow the network to learn increasingly complex features from the input data.
COMPONENT 02

Neurons

Neurons are the basic computational units of a neural network. They function similarly to neurons in the human brain by receiving inputs, processing them, and generating outputs.

How Neurons Work:

  1. Receive input values.
  2. Multiply inputs by weights.
  3. Add a bias value.
  4. Apply an activation function.
  5. Produce an output.
Role: Neurons identify relationships and patterns in data. output = σ(Σ(wᵢxᵢ) + b)
COMPONENT 03

Weights

Weights are numerical values that determine the strength and importance of connections between neurons.

Role:
  • Large weights indicate stronger influence.
  • Small weights indicate weaker influence.

During training, the neural network continuously adjusts weights to improve prediction accuracy.

COMPONENT 04

Activation Functions

Activation functions are mathematical formulas that determine whether a neuron should activate and pass information forward.

Common Activation Functions:

  • Tanh: Produces outputs between -1 and 1.
  • ReLU (Rectified Linear Unit): Outputs positive values while ignoring negatives.
  • Sigmoid: Produces outputs between 0 and 1.
Role: Activation functions introduce nonlinearity, allowing neural networks to learn complex patterns and relationships.
COMPONENT 05

Loss Functions

Loss functions measure how far the model's predictions are from the correct answers.

Examples:

  • Mean Squared Error (MSE): Common in regression problems.
  • Cross-Entropy Loss: Common in classification problems.
Role: The loss function guides the network during training by showing how much error exists.
  • Low loss = better predictions.
  • High loss = poorer predictions.
COMPONENT 06

Optimization Algorithms

Optimization algorithms adjust the network's weights to minimize the loss function and improve performance.

Common Algorithms:

  • Gradient Descent
  • Stochastic Gradient Descent (SGD)
  • Adam Optimizer
Role: Optimization algorithms help the model learn efficiently by finding the best weight values for accurate predictions. w ← w - η · ∂L/∂w

Training Process: How the Network Learns

📥
Step 1 — Forward Pass

Input features (X₁, X₂ from Playground) flow through each layer. Each neuron applies tanh(Σwᵢxᵢ + b) and passes its output forward. The final layer produces a prediction ŷ.

📊
Step 2 — Loss Calculation

The prediction is compared to the true label. In Experiment 1 (epoch 0), loss = 1.584 — the model was guessing. The loss function quantifies exactly how wrong the current weights are.

🔄
Step 3 — Backpropagation

The chain rule is applied in reverse to compute how much each weight contributed to the loss (∂L/∂w). This gives each weight a gradient — direction and magnitude for improvement.

✅
Step 4 — Weight Update (SGD, lr = 0.03)

Each weight is nudged: w ← w − 0.03 · ∂L/∂w. After 284 epochs (Experiment 2), loss reached 0.000. The loop repeats every epoch until convergence.

Visualizing neural networks bridges the gap between abstract mathematics and intuitive understanding. By watching the Playground in real time — loss dropping, boundaries sharpening, weights strengthening — the mechanics of machine learning become tangible rather than theoretical.

📉

🧱
Match Depth to Complexity

Gaussian needed 0 layers; circle also needed 0 — with the right features. Deeper networks are for more complex datasets, not all datasets.

🔀
Non-Linearity is Essential

Activation functions give networks expressive power beyond linear models. Without them, stacked layers solve nothing new.

🎯
Feature Engineering = Power

X₁²/X₂² solved the circle without any hidden layers. Thoughtful feature selection can replace network depth.

Weights Encode Knowledge

All intelligence lives in learned weight values — adjusted iteratively via backpropagation to reduce loss.

🎲
Noise Tests Generalization

Experiment 2 used noise=5 yet converged — because the Gaussian problem is inherently simple and linearly separable.

1 · Theoretical Background

1.1 Supervised Learning

Trains a model on labeled data where each input has a known correct output. The model minimizes the difference between predictions and true labels.

  • Data: labeled (input + correct output)
  • Goal: predict output for new inputs
  • Examples: Classification, Regression
  • Evaluation: clear via loss / accuracy
1.2 Unsupervised Learning

Uses unlabeled data — the model must find hidden structure on its own with no predefined correct answers.

  • Data: unlabeled (input only)
  • Goal: discover patterns/groupings
  • Examples: K-Means, PCA
  • Evaluation: harder to measure objectively
1.3 Loss Functions

Measures how wrong predictions are. The optimizer minimizes this by adjusting weights — lower loss = better predictions.

  • MSE — regression; penalizes large errors heavily
  • Cross-Entropy — classification; measures probability divergence
  • TF Playground shows train & test loss in real-time
1.4 Parameter Tuning & Fitting

Adjusting settings that control how a model learns. Overfitting: low train loss, high test loss. Underfitting: both losses remain high.

  • Learning Rate — step size per update
  • Neuron Count — model capacity
  • Hidden Layers — hierarchical features
  • Activation — non-linearity (Tanh, ReLU)
  • Regularization — prevent overfitting (L1/L2)
